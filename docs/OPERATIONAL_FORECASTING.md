# Operational forecasting

Phase 4 Step 2 adds advisory operational forecasts derived from runtime truth:

- `operational_forecasting`, `runtime_risk_projection`, `scalability_forecasts`, `enterprise_operational_outlook`

Forecasts are bounded, explainable, and never trigger autonomous execution.

**API:** `GET /api/v1/mission-control/runtime/forecasts`, `GET …/runtime/outlook`  
**CLI:** `aethos runtime forecasts`, `aethos runtime outlook`
