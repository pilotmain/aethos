# Operational storytelling (final)

Step 13 completes operator-facing narratives without exposing raw engine internals.

## Surfaces

- `operational_storytelling_final` on runtime truth
- `GET /api/v1/mission-control/runtime-story` (Step 9+)
- Office command center — health-first summary

## Narrative elements

Operators see:

1. **What is happening** — recovery / pressure / routing summary
2. **Why it matters** — orchestrator accountability
3. **What changed** — bounded shift list from narratives v2
4. **What needs attention** — escalations, fallback routing
5. **What AethOS is doing** — coordination, governance, calm operations

Copy is calm, advisory-first, and bounded.
