# Nexa — User Guide

**Nexa is your personal execution system.**

It doesn’t just answer questions —  
it helps you **think, decide, and get things done**.

---

## What Nexa does

Most tools stop at conversation.

Nexa goes further:

- Turns ideas into **clear plans**
- Breaks work into **actionable steps**
- Helps you **execute with agents**
- Tracks progress across sessions
- Generates documents, research, and decisions

---

## How it works

You talk to Nexa like a normal chat:

> “Analyze this website and tell me how to improve messaging”

Nexa responds with:

- Structured thinking
- Clear insights
- Actionable next steps

Then you can say:

> “Do the first one”

And Nexa continues the work.

---

## What makes Nexa different

Nexa is not:

- a chatbot  
- a prompt tool  
- a UI-heavy dashboard  

Nexa is:

> A **system that connects thinking → decision → execution**

---

## Core capabilities

- Multi-session workspaces  
- Built-in and custom agents  
- Web research (read-only + search)  
- Document generation (PDF, Word, Markdown)  
- Task and job execution system  
- Cost and usage awareness  
- Lightweight workflows (no setup required)  

---

## Example

**You:**

> “Create a marketing plan for my SaaS product”

**Nexa:**

- Analyzes the problem  
- Structures a plan  
- Suggests next steps  

**You:**

> “Run the research”

**Nexa:**

- Uses web tools  
- Summarizes findings  
- Updates the plan  

---

## Why Nexa exists

Most tools make you:

- think manually  
- organize manually  
- execute manually  

Nexa reduces that gap.

---

## Getting started

1. Run Nexa locally  
2. Open the web UI or Telegram  
3. Start with a goal:  
   - “Plan my week”  
   - “Analyze this site”  
   - “Create a proposal”
