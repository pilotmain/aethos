# Contributing (docs)

For the full contributor guide, see the repository root **[CONTRIBUTING.md](../../CONTRIBUTING.md)**.

Short version:

1. Fork or branch, keep changes focused.
2. Run **`pytest`** and **`python -m compileall -q app`** before pushing.
3. Match existing style; use **`ruff check app tests`** where applicable.

Apache 2.0 applies to contributed code (see root `LICENSE`).
