# Command execution

## Running commands

Examples users may try:

- “Run `ls -la`”
- “Execute `npm install`”

## Installing packages

Examples:

- “Install express”
- “`npm install react`”
- “`pip install requests`”

## Security

- **Allowlists** for host and sandbox executors (no arbitrary shell in sandbox v1).
- **Workspace isolation** for sandboxed plans.
- **Approvals** for risky or privileged operations — see [DEV_JOBS.md](../DEV_JOBS.md) and [OPERATIONS.md](../OPERATIONS.md).
